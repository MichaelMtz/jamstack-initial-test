Build tooling

The HTML versions of the plans are generated from their markdown sources. Edit the .md, re-run the corresponding script, and the matching .html is regenerated. Both build scripts use the same shared template and library, so they stay visually consistent.

  • jsscripts/build-one-page-html.js
  • jsscripts/build-deployment-html.js
  • jsscripts/build-executive-plan-html.js
  • jsscripts/lib/build-doc.js — shared md→html builder (multi-section docs)
  • htmlscripts/templates/doc-template.html — shared shell for sidebar+bookmarks docs
  • htmlscripts/templates/one-page-template.html — print-friendly one-pager shell
How to rebuild the HTML docs 
cd src/assets/ai-chat-concept

node scripts/build-one-page-html.js
# → writes ai-chat-one-page.html

node scripts/build-deployment-html.js
# → writes deployment-plan.html

node scripts/build-executive-plan-html.js
# → writes ai-chat-executive-plan.html
